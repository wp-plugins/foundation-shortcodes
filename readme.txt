Adds WordPress shortcode support for Foundation layouts and UI elements - to be used with themes built with Foundation 5.

== Description ==

This plugin adds WordPress shortcode support for Foundation 5 layouts and UI elements, including rows, columns, block grids, buttons, tabs, accordions and modals. This plugin is meant to be used with themes built on Foundation 5.

Full usage examples and documentation can be seen below:

[Foundation Shortcodes Documentation](http://www.oyova.com/foundation-shortcodes.html)

== Installation ==

1. Unzip and upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Can I use this if my theme isn't built with Foundation 5? =

Some elements will work if the theme is built with Foundation 4, but other than that, no.
